# Security

*   **Certificates:** Manage TLS certificates for secure communication.
*   **Kubeconfig:** Configure access to clusters.
*   **API Groups:** Organize Kubernetes APIs.
*   **Authorization Modes:**
    *   **RBAC (Role-Based Access Control):** Recommended method for managing access.
    *   **Node, ABAC, Webhook:** Other authorization modes.
*   **Service Accounts:** Provide an identity for pods.
*   **ImagePullSecrets:** Provide credentials to pull images from private registries.
*   **Network Policies:** Control traffic flow between pods.
